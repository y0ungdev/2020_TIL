Django html에서 공통적인 부분에 대해서 사용 가능한 `base.html` 생성 후 `settings.py`에 등록해야 함

공통되는 부분이 아닌 내용 부분은 `block`요소를 이용할 수 있음

```html
{% block content %}	# content 적는 것 필수
{% endblock %}		# 여기는 content 적어도/안 적어도 무관
```
